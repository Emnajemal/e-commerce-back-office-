Basic Date

Use ng2-flatpickr component to get basic flatpickr.

Date & Time

Use enableTime: true option to make date and time flatpickr.

Default Date

Use defaultDate: '2019-03-19' option to get a default date.

Date Range

Use mode: 'range' option to select range of dates.

Time

Use enableTime: true, noCalendar: true option to only get time.

Custom Date

Use altInput: true, altFormat: 'j-m-Y' option to customize time format.

Multiple Date

Use mode: 'multiple' option to select multiple of dates.
